List published crate releases with version, yanked/prerelease status, publication timestamp, and declared Rust version when crates.io provides it.

Use this for release history and choosing upgrade boundaries. By default yanked and prerelease versions are excluded. Set `check_docs=true` when rustdoc JSON availability matters; this performs an additional docs.rs status lookup per returned version. Use `resolve_version` when only one matching version is needed.
